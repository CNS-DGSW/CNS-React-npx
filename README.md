# CNS React + typescript + vite BoilerPlate

React + vite + typescript eslint로 구성되어 있는 BoilerPlate 입니다

CNS 동아리 내에서 프로젝트를 할 때마다 폴더구조를 세팅해야 하는 귀찮음을 덜고자 만들게 되었습니다.


# 실행방법
`npx cns-react-app 프로젝트이름`
### 폴더구조

```
├─public
└─src
    ├─assets
    ├─components
    ├─hooks
    ├─pages
    ├─repository
    ├─types
    └─util
```
